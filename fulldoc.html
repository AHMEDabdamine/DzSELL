1. Configuration

The first step is to configure the store at the Office module. Please refer to the Configuration wiki page to learn how to do that.

Older versions may use the StoreConfig module instead. See the following:

  • MainScreen -> StoreConfig -> click

2. Employees

You'll need to create employee users who can log in to the store. Starting in version 3.2.0, the Employee module is found within the Office module.

  • MainScreen -> Office -> (click) -> Employee -> (click)

Please refer to the Employees wiki page for detailed instructions.

2.1. Permissions

Remember to set appropriate permissions for your employees to manage inventory.

3. Inventory

This module allows you to load Items and Kits. It's a complex module, but Dz SELL is well-integrated. For example, you can create new inventory items directly from the Sales module.

3.1. Items

To load a product into the inventory, click the Items button. This will display your existing items. See the Inventory Items wiki page for more information.

3.2. Kits

You can group inventory items into Kits to sell them as combo products. Click the Items Kits module button to manage kits. Refer to the Inventory Kits wiki page for guidance.

4. Sales

This module allows you to process Sales and Returns. It's a comprehensive module with many features, including different payment methods, receipt printing, and a temporary item feature.

4.1. Sales Modes

Dz SELL includes a Cash Register / Point of Sale module. See the Sales Modes wiki page for details.

4.2. Sales Restaurant

With a few adjustments, Dz SELL can work as a POS system for restaurants using the Table feature. See the Sales Restaurant wiki page for information.

5. Expenses

Dz SELL has a basic Expenses module for simple expense tracking. This module does not include accounting functionality.

Please refer to the Expenses wiki page for details on using this module.

Point of Sale (POS)
  • Sell products or/and services
  • Apply multiple payments (including different types)
  • Featured Restaurant Tables module
  • Purchase Orders
  • Advanced Transaction Types (Special Orders, Back Orders, Layaways)
  • Credit accounting sales

  • Add customers
  • Import/export from CSV
  • Maintain customer profiles
  • View full sales history

  • Accounts Receivable / House Accounts

Inventory Management
  • Create stock and non-stock items with custom fields
  • Import/export from CSV
  • Generate and read barcodes
  • Inventory Matrix
  • Track item features, colors, materials
  • Support QR codes
Multilingual Web Interface (i18n GUI)
  • Multilingual support
  • Selectable Bootstrap (Bootswatch) based UI themes
  • End-user customize interface like WordPress (with minimal coding)
  • Regionalization per item attributes
Reporting
  • Customers, Inventory, and Transactions (sales or returns)
  • Dashboard with Top Sales, Top Suppliers, Best Selling Items, Top Customers
Expense Modules
  • Add and summarize basic expenses
  • Accounting with expenses over sales
Gift Card and Rewards
  • Issue gift cards as a payment method
  • Customer rewards
  • Management of accounting gift cards
  • Expired dates on rewards
